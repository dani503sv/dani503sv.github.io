Operador Ternario

//opcion para in if-else siemple y rapido
const saludo = nombre === "daniel" ? "Hola Daniel" : "Hola desconocido";
//condicion    if    if    else
Tip:
1. Prioriza condiciones más probables Coloca primero las condiciones que más probablemente se cumplan para mejorar el rendimiento.
2. Evita condicionales complejos Divide condiciones largas en variables intermedias con nombres claros
3. Usa switch para múltiples opciones Cuando compares muchas opciones con una sola variable, elige switch en lugar de muchos if.
4. Ternario para condiciones simples Ideal para asignaciones cortas.
5. Evita condicionales innecesarias No pongas == true o == false si ya tienes una expresión booleana.
6. Agrupa condiciones relacionadas Usa paréntesis para evitar errores de lógica y mejorar la lectura.
Ciclos While && For

Ciclos

FOR
//ejemplo de for
//Se usa cuando se sabe cuando finaliza
for(let i = 0; i<=4; i++){
console.log("FOR Se repite 5 veces, inicia de 0 :" + i )
}
WHILE
//ejemplo de while
//se usa cuando no se sabe cuando finalizara
//se usa con un contador o bandera
let contador = 0;
while(contador <= 4 ){
console.log("WHILE Se repite 5 veces, inicia de 0 :" + contador )
contador++;
}
DO-WHILE
//ejemplo de do-while
//Entra solo 1 vez y luego evalua el condicional
//usado con contadores o banderas
contador = 0;
do{
console.log("DO-WHILE Se repite 5 veces, inicia de 0 :" + contador )
contador++;
}while(contador <= 4 );
Tip:
1. Conoce el tipo de bucle ideal para cada caso
Usa for cuando conoces la cantidad de iteraciones.
Usa while cuando no sabes cuántas veces se repetirá (condición depende de algo externo).
Usa foreach para recorrer colecciones sin preocuparte por los índices.
2. Cuidado con bucles infinitos
