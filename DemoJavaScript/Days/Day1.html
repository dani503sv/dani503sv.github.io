VARIABLES & CONSTANTES

Declaración y ámbito

Declaración y ámbito
// Usar let (ámbito de bloque {})
let nombre = "Daniel";

// var tiene ámbito global (evitar)
var variableAntigua = "Evitar esto";

// const nunca cambia su valor
const PI = 3.1416;
Mejor práctica: Prefiere siempre let sobre var para mejor ámbito. Usa const para valores que no deben cambiar.
Trabajando con objetos
let objeto = {
  nombre: "Daniel amaya",
  edad: 21
};

// Añadir nueva propiedad
objeto.correo = "dani@gmail.com";

// Arrays (que son objetos)
const colores = ["verde", "naranja", "amarillo"];
colores.push("uno mas"); // Añadir nuevo elemento
Nota: Aunque const evita la reasignación, las propiedades de objetos y elementos de arrays pueden modificarse.

TIPOS DE DATOS & MÉTODOS

Strings y métodos comunes

Los strings pueden usar comillas simples (') o dobles ("):

let nombre = "Daniel";
let apellido = 'Amaya';
Métodos para strings
  • .length - Obtener longitud
    nombre.length // Retorna 6
  • .includes() - Buscar substring
    nombre.includes("Dan") // Retorna true
  • .trim() - Eliminar espacios
    " hola ".trim() // Retorna "hola"
  • .split() - Dividir en array
    "a,b,c".split(",") // Retorna ["a", "b", "c"]
Recuerda: Los strings en JavaScript son inmutables. Métodos como trim() retornan nuevos strings en lugar de modificar el original.
